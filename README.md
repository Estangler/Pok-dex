# 🔥 Pokédex - React + TypeScript

Uma Pokédex interativa consumindo a API pública PokéAPI, desenvolvida com foco em boas práticas de front-end, performance e experiência do usuário.

---

## 🧠 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Consumo de APIs REST
* Manipulação de dados assíncronos
* Gerenciamento de estado com React
* Tipagem com TypeScript
* Estruturação de código pensando em escalabilidade

A aplicação permite buscar Pokémons por nome, visualizar seus tipos e exibir informações básicas como ID e imagem.

---

## ⚙️ Tecnologias utilizadas

* React
* TypeScript
* TailwindCSS
* Fetch API

---

## 🔍 Funcionalidades

* 🔎 Busca de Pokémon por nome
* 🖼️ Exibição de imagem (sprite)
* 🏷️ Tipagem dinâmica (cores baseadas no tipo)
* 🔢 Formatação de ID (#001, #025, etc)
* ⚡ Consumo de dados com `Promise.all`
* 🎯 Filtro em tempo real

## ⚡ Como rodar o projeto

```bash

# Entre na pasta
cd pokedex

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
